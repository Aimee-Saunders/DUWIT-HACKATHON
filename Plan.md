### DUWiT Scheduler
The problem – we spend more time organising our lives than actually working/doing activities, we want this process to be organised and intelligently scheduled for us.
Our solution – a calendar style web app to organise student life, Durham themed, main feature is an algorithm to organise event blocks efficiently
## Features
-	Ability to add events that must occur at a strict time (hard) e.g. lectures, meetings
-	Ability to add events that can occur at any time (soft) e.g. revision slots, meals
-	Algorithm to schedule soft events around hard events
-	AI plug in – for what to do with blocks of free time?
-	F1 car progress tracker
## Algorithm
-	Should take forgetting curve into account
-	Pomodoro technique (maybe an option to select a different technique)
